Introducción a SVG

SVG (Scalable Vector Graphics) es un formato gráfico basado en XML utilizado para definir gráficos vectoriales en la web. SVG permite crear gráficos que pueden ser escalados sin pérdida de calidad, lo que lo convierte en una excelente opción para elementos visuales en páginas web.

En esta sección, verás ejemplos de cómo usar SVG para crear gráficos y animaciones.

Crear un círculo en SVG

El siguiente ejemplo muestra cómo crear un círculo utilizando SVG. La etiqueta <circle> define un círculo con un radio y una posición central.

<svg width="200" height="200">
    <circle cx="100" cy="100" r="50" stroke="black" stroke-width="2" fill="red" />
</svg>

Crear un rectángulo en SVG

La etiqueta <rect> permite crear un rectángulo con ancho, alto, bordes redondeados y otros atributos. Aquí te mostramos un ejemplo de un rectángulo con un borde redondeado.

<svg width="200" height="200">
    <rect x="50" y="50" width="150" height="100" rx="15" ry="15" style="fill:blue;stroke:black;stroke-width:2" />
</svg>

Crear una línea en SVG

Con la etiqueta <line>, puedes dibujar líneas entre dos puntos. A continuación te mostramos cómo crear una línea diagonal.

<svg width="200" height="200">
    <line x1="0" y1="0" x2="200" y2="200" style="stroke:black;stroke-width:2" />
</svg>

Crear un texto en SVG

La etiqueta <text> te permite agregar texto a tus gráficos SVG. A continuación se muestra cómo agregar texto con estilo y posición personalizada.

<svg width="200" height="200">
    <text x="50" y="100" font-family="Verdana" font-size="35" fill="black">Hola SVG!</text>
</svg>

Uso de path en SVG

La etiqueta <path> permite dibujar formas más complejas usando comandos. A continuación, un ejemplo de cómo dibujar un corazón utilizando un path.

<svg width="200" height="200">
    <path d="M10 80 C 40 10, 65 10, 95 40 S 150 80, 95 120 C 65 140, 40 130, 10 110 Z" fill="red" />
</svg>
