Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. Permite controlar el diseño, los colores, las fuentes y otros aspectos visuales de una página web.

En esta sección, verás ejemplos de cómo utilizar propiedades CSS para dar estilo a los elementos HTML.

Propiedad <color> - Cambiar color de fondo

La propiedad background-color permite cambiar el color de fondo de un elemento HTML.

body {
    background-color: lightblue;
}

Resultado en el navegador:

Este es un ejemplo con el fondo de color lightblue.

Propiedad <font-family> - Cambiar la fuente

La propiedad font-family se usa para especificar la fuente de texto que se va a utilizar en un elemento HTML.

h1 {
    font-family: 'Arial', sans-serif;
}

Resultado en el navegador:

Este es un título con la fuente Arial

Propiedad <margin> - Márgenes alrededor de los elementos

La propiedad margin se usa para crear espacio alrededor de los elementos. Es útil para separar los elementos entre sí.

p {
    margin: 20px;
}

Resultado en el navegador:

Este es un párrafo con un margen de 20 píxeles alrededor.

Propiedad <border> - Bordes alrededor de los elementos

La propiedad border se utiliza para agregar bordes alrededor de los elementos. Puedes especificar el grosor, el estilo y el color del borde.

div {
    border: 2px solid black;
}

Resultado en el navegador:

Este es un div con un borde de 2 píxeles, sólido y de color negro.

Propiedad <display> - Controlar la visualización de los elementos

La propiedad display se usa para controlar cómo se muestran los elementos HTML. Un valor común es inline o block.

span {
    display: inline-block;
    background-color: yellow;
    padding: 5px;
}

Resultado en el navegador:

Este es un span con display: inline-block.

Propiedad <width> y <height> - Controlar el tamaño de los elementos

Las propiedades width y height permiten definir el ancho y la altura de los elementos.

img {
    width: 100px;
    height: auto;
}
