Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Está compuesto por una serie de etiquetas que estructuran el contenido en la web. En esta sección, aprenderás a utilizar algunas de las etiquetas más importantes en HTML.

Etiqueta <h1> a <h6> para encabezados

Las etiquetas de encabezado se usan para estructurar el contenido de una página web, representando títulos de diferentes niveles. El <h1> es el título más importante y el <h6> es el de menor jerarquía.

<h1>Este es un título principal</h1>
<h2>Este es un título secundario</h2>
<h3>Este es un título terciario</h3>

Resultado en el navegador:

Este es un título principal

Este es un título secundario

Este es un título terciario

Etiqueta <p> para párrafos

La etiqueta <p> se usa para definir párrafos de texto en HTML.

<p>Este es un párrafo de texto</p>

Resultado en el navegador:

Este es un párrafo de texto.

Etiqueta <img> para imágenes

La etiqueta <img> se utiliza para insertar imágenes en la página web. A continuación, mostramos un ejemplo con un atributo de fuente de imagen.

<img src="https://via.placeholder.com/150" alt="Imagen de ejemplo">

Resultado en el navegador:

Etiqueta <a> para enlaces

La etiqueta <a> se usa para crear enlaces a otras páginas web o recursos. A continuación se muestra cómo vincular a otro sitio.

<a href="https://www.w3schools.com">Visitar W3Schools</a>

Resultado en el navegador:

Visitar W3Schools

Etiqueta <ul> para listas desordenadas

La etiqueta <ul> se utiliza para crear listas no ordenadas (con viñetas).

<ul>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>

Resultado en el navegador:

  • Elemento 1
  • Elemento 2
  • Elemento 3
